# 🚗 Road Accident Analysis & Forecasting

## 📌 Overview
This project is based on my published research paper on road accident analysis. 
It focuses on identifying key factors contributing to accidents and applying 
data-driven techniques for better road safety insights.

## 🔗 Research Paper
Read full paper here:
https://journals.sagepub.com/doi/10.1177/15741699261432601

## 🎯 Objectives
- Analyze road accident patterns
- Identify key contributing factors
- Perform statistical and time-series analysis
- Provide actionable safety insights

## 📊 Key Insights
- Human factors (speeding, distraction) are major contributors
- Road and environmental conditions impact severity
- Data-driven methods help predict accident trends

## 🛠️ Tech Stack
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Statsmodels (for forecasting)

## 📈 Features
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Visualization of accident trends
- Forecasting using statistical models
